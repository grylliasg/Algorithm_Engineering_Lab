Για την εκτέλεση των αλγορίθμων στο diogenis ακολουθήστε τα βήματα:


1. Μεταφερθείτε στον φάκελο bin

2. ~ Για το compile της δομής bridge-blocks χρησιμοποιήστε την εντολή make -f Makefile.bridge compile,
   ενώ για το run make -f Makefile.bridge run

   ~ Για το compile της δομής blocks χρησιμοποιήστε την εντολή make -f Makefile.blocks compile,
   ενώ για το run make -f Makefile.blocks run

3. Ανάλογα με τον τύπο του πειράματος που θέλετε να εκτελέσετε, χρησιμοποιήστε και την κατάλληλη main 
   βγάζοντας τη από τυχόν σχόλια.
